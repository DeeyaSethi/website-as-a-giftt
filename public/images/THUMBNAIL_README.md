# Music Thumbnail Setup

## Default Thumbnail

Place your default music thumbnail here as:
**`music-thumbnail-default.png`**

### Requirements:
- **Format:** PNG or JPG
- **Size:** 1920 x 1080 pixels (16:9 ratio)
- **File name:** `music-thumbnail-default.png`

### Current Default:
The "Music Made with Love" thumbnail with:
- Pink romantic gradient background
- YouTube logo at top
- "Music Made with Love" title
- "A HEARTFELT PLAYLIST" subtitle
- Professional, generic design

---

## How It Works:

1. **YouTube embedding works:** Normal player shows
2. **YouTube embedding disabled:** Shows this default thumbnail
3. **User provides custom thumbnail:** Uses their URL instead

---

## To Update Default Thumbnail:

1. Create/design new thumbnail (1920x1080)
2. Save as `music-thumbnail-default.png`
3. Replace file in this directory
4. Done! All music templates will use new default
